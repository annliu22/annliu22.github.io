FPGA Dynamic Music Visualizer

The FPGA Dynamic Music Visualizer takes live MIDI instrument inputs and visualizes the corresponding sheet music live. The written music loops, allowing the user to layer music over what has already been recorded.

PSoC Paint

Microsoft Paint (MS Paint) is emulated using Cypress's PSoC, resulting in a basic drawing program with hardware inputs and a VGA output to a monitor.

ProKey8: Custom Keyboard for Procreate

The "ProKey8" is a custom, bluetooth wireless keyboard for Apple's Procreate drawing app; its ergonomic and tactile form enable a more efficient drawing process.
